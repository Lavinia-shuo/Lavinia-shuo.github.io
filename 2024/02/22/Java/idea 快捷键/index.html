Idea小tips

需要创建新的类时，可以直接在目录中复制粘贴代码，不需要先新建并命名类再粘贴。

自动生成对应方法：alt + 回车

从service接口跳转到service实现类：alt+ control +点击service接口名

查看方法注释：ctrl + q
