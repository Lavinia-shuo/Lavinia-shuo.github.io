Git

版本管理 git log：查看当前分支的所有版本 git reset --hard 版本号: 回滚到某个版本（在本地就能实现）

新项目如何上传到github ①在github上，新建一个仓库 ②本地bash : git remote add origin git@github.com:xxx/XXX.git：将本地仓库关联到远程仓库
git push --set-upstream master ：如果没有这个分支 则创建

分支 git branch：查看所有分支和当前所处分支 git checkout -b branch_name：创建并切换到branch_name这个分支 git checkout branch_name：切换到branch_name这个分支 git merge branch_name：将分支branch_name合并到当前分支上 git push --set-upstream origin branch_name : 推送分支时，如果云端没有branch_name这个分支，则在云端创建它

git branch -d branch_name：删除本地仓库的branch_name分支（需要先切换到其他分支） git push -d origin branch_name：删除远程仓库的branch_name分支

注意 1、AC Terminal 是一个终端，可以登录到任何一个server。课程给每个人分配了一个远程server 2、云端分支和本地分支相互独立
